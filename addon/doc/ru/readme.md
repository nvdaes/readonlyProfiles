# Профили только для чтения

* Автор: Noelia Ruiz Martínez

Если это дополнение включено, профили конфигурации не будут сохраняться автоматически.

Нажмите NVDA+shift+p, чтобы сохранить текущую конфигурацию в самый последний профиль.

Эту команду можно изменить в меню NVDA, подменю параметров, диалоге жестов ввода, в категории конфигурации.

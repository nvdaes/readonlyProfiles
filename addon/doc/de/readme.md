# Schreibgeschützte Profile

* Autorin: Noelia Ruiz Martínez

Wenn dieses Addon aktiviert ist, werden Konfigurationsprofile nicht automatisch gespeichert.

Drücken Sie NVDA+Umschalt+p, um die aktuelle Konfiguration im neuesten Profil zu speichern.

Dieser Befehl kann im NVDA-Menü, im Untermenü "Optionen" im Dialogfeld "Tastenbefehle“ und in der Kategorie "Konfiguration" geändert werden.

# Ei-muokattavat profiilit

* Tekijä: Noelia Ruiz Martínez

Asetusprofiileja ei tallenneta automaattisesti, kun tämä lisäosa on käytössä.

Tallenna nykyiset asetukset viimeksi käytettyyn profiiliin painamalla NVDA+Vaihto+P.

Tätä komentoa on mahdollista muuttaa NVDA-valikon kohdasta Mukautukset -> Näppäinkomennot -> Asetukset-kategoria.

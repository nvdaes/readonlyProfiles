# Profili samo za čitanje

* Autor: Noelia Ruiz Martínez

Kad je ovaj dodatak omogućen, konfiguracijski profili neće se automatski spremati.

Pritisnite `NVDA+Shift+P` za spremanje trenutne konfiguracije u najnoviji profil.

Ova komanda može biti promijenjena kroz NVDA izbornik, podizbornik "Opcije", dijalog ulaznih gesti u kategoriji "Konfiguracija".
